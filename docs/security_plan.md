# Klino Security Plan

## Goal

Create a safe and trusted environment for customers and cleaning agents.

---

# Customer Safety

## Account Verification

Customers provide:

- Full name
- Phone number
- Email
- Address

Optional:

- Identity verification

---

# Agent Safety Verification

Agents provide:

- Government ID
- Profile photo
- Phone number
- Address
- Bank details

Agent account status:

- Pending Verification
- Verified
- Suspended

Only verified agents can receive bookings.

---

# Booking Safety

Every booking has:

- Unique Booking ID
- Customer details
- Agent details
- Date and time
- Service location

---

# Cleaning Evidence

Before cleaning:

Agent uploads:

- Before photos
- Arrival confirmation

After cleaning:

Agent uploads:

- After photos
- Completion confirmation

---

# Payment Protection

Customer payment is held securely until:

1. Agent completes the cleaning
2. Agent uploads completion evidence
3. Customer confirms completion

---

# Dispute Protection

If there is a problem:

Customer can:

- Report issue
- Upload evidence
- Contact support

Agent can:

- Respond to dispute
- Provide evidence

Admin reviews the case.

---

# Privacy

Klino protects:

- User information
- Documents
- Payment information
- Photos

Access is limited based on user roles.
