# Klino Website Pages

## Goal

Define all pages required for the Klino platform.

---

# Public Pages

## 1. Home Page

Purpose:

Introduce Klino to visitors.

Sections:

- Klino logo
- Main message
- Book Cleaning button
- Become an Agent button
- Why Choose Klino
- How Klino Works
- Safety Information
- Footer

---

## 2. About Klino

Purpose:

Explain:

- Company mission
- Vision
- Values
- Commitment to safety

---

## 3. How It Works

Explain:

Customer:

1. Create account
2. Choose service
3. Book cleaner
4. Confirm completion
5. Rate service

Agent:

1. Register
2. Get verified
3. Accept jobs
4. Complete cleaning
5. Receive payment
