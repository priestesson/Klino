# Klino Agent Dashboard Design

## Goal

Give verified cleaners a place to receive jobs, manage cleaning assignments, upload proof of work, and track earnings.

---

# Agent Dashboard

## Header

Welcome, [Agent Name]

Verification Status:
- Verified ✅
- Pending Verification ⏳

---

# New Booking Requests

Display:

- Customer Name
- Location
- Service Type
- Date
- Time
- Estimated Payment

Actions:

[Accept Job]

[Decline Job]

---

# Active Job

When an agent accepts a booking:

Show:

Customer Details

Cleaning Address

Service Details

Booking ID

Scheduled Time


Actions:

[Start Job]

---

# Before Cleaning

When agent arrives:

Agent uploads:

- Before Photos
- Arrival Confirmation

Optional:
- GPS Check-in

---

# During Cleaning

Status:

Cleaning In Progress

---

# After Cleaning

Agent uploads:

- After Photos

Then:

[Mark Job Complete]

---

# Earnings

Show:

- Completed Jobs
- Pending Payments
- Total Earnings
- Payment History

---

# Profile

Agent can manage:

- Personal Details
- Profile Photo
- Documents
- Bank Details
- Availability
