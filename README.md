# Klino

Klino is a trusted cleaning services platform that connects customers with verified professional cleaners.

## Vision

To become Africa's most trusted platform for booking safe, reliable, and professional cleaning services.

## Features

- Customer registration and login
- Agent registration and verification
- Secure booking system
- Escrow payments
- Before & after cleaning photos
- Ratings and reviews
- Admin dashboard
- Notifications
- Customer support

## Tech Stack

- Frontend: Flutter
- Backend: Python (FastAPI)
- Database: PostgreSQL
- Git: GitHub
